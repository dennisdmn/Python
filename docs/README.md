# Docs

Documentacao geral do repositorio.

## Estrutura

- `casos_sucesso/`: registros praticos de problemas resolvidos, com contexto, abordagem e resultado
- `notas_tecnicas/`: anotacoes objetivas sobre bibliotecas, recursos e tecnicas
- `comparativos/`: comparacoes entre ferramentas, engines e abordagens
- `referencias/`: links, resumos e materiais de apoio
