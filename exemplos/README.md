# Exemplos

Area para exemplos curtos, didaticos e autocontidos.

Use esta pasta para consultar rapidamente uma tecnica isolada, comparar abordagens simples e reaproveitar pequenos blocos em projetos reais.

## Organizacao

- `excel/`: exemplos de leitura e processamento de planilhas.
- `windows/`: exemplos voltados a automacao local no Windows.

## Criterios para novos exemplos

- Manter exemplos pequenos e focados em uma unica tecnica.
- Preferir nomes claros, indicando biblioteca e objetivo.
- Incluir comentarios somente onde ajudam a entender a decisao tecnica.
- Quando o exemplo depender de uma plataforma especifica, separar em uma pasta propria, como `windows/`.
