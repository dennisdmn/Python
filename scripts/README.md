# Scripts

Area para scripts reutilizaveis e mais proximos de uso operacional.

## Sugestao de organizacao

- `excel/`: leitura, exportacao e utilitarios para planilhas
- `sqlite/`: criacao, carga e consulta em SQLite
- `arquivos/`: manipulacao de arquivos e diretorios
- `automacao/`: rotinas automatizadas
- `utilitarios/`: funcoes e scripts de apoio geral
